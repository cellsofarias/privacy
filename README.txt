ESTRUTURA DO SITE CELLSO

1. Envie todo o conteúdo desta pasta para a raiz do repositório.
2. Mantenha o arquivo CNAME que você já possui no GitHub.
3. Substitua os blocos escritos ADICIONAR LOGO/PRINT/VÍDEO pelos arquivos reais.
4. Para usar imagens, coloque-as na pasta /ativos e troque o bloco por algo como:
   <img src="/ativos/nome-do-arquivo.png" alt="Descrição">
5. Para vídeo do YouTube, substitua o bloco pela tag iframe fornecida pelo YouTube.
6. As URLs limpas funcionarão com esta estrutura:
   /minha-empresa
   /contato
   /privacidade
   /termos
7. Depois de testar, você pode apagar os arquivos antigos contato.html, privacidade.html e termos.html.
